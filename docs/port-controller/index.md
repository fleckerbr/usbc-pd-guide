# Type-C Port Controller

A Type-C Port Controller (TCPC) is a dedicated piece of hardware that exists to semi-autonominously handle some aspects of a USB-C connection. This section of the guide will discuss how a Type-C Port Manager (TCPM) is intended to interact with a TCPC via the Type-C Port Controller Interface (TCPCI).

Additionally, since the separation between a Type-C Port Manager (TCPM) and a TCPC isn't defined by the USB-C Cable and Connector Specification this section is also intended to help clarify what actions need to be handled on each side of the interface.
